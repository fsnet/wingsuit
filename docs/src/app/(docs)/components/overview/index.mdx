# Overview

The following component types are available with wingsuit:
* ### Wingsuit component
Use this component in most cases. Read [here for more information](/components/wingsuit).
* ### Wingsuit component with presentation template
A component if you need an additional [presentation template](/components/presentation).
* ### Twig only component
Use [Twig only](/components/twig) if you have a very simple template without any data.
* ### Twig only component with presentation template
The same as a Twig template with an additional presentation template.
* ### Presentation template
A presentation template only. Choose this if you want to build a complete Page.
