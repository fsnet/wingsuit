## Prerequisites

- [Node `^12`](https://nodejs.org)
- [YARN `^1.22`](https://classic.yarnpkg.com/) OR [NPM `^6.10`](https://docs.npmjs.com/)

- [PHP `^7.0.0`](https://php.net)

Get started using the automated command line tool. This command creates a Wingsuit demo project, which uses the [atomic design guidelines](https://bradfrost.com/blog/post/atomic-web-design/) to structure the patterns.
